### 上手指南

###### 开发前的配置要求

1. jdk 8.0
2. tomact 9.0.20
3. maven 3.6.3
4. mysql 5.6
5. vue 2.9.6
6. intellij idea 2020.1.1
7. node 16.13.0

###### **安装步骤**

1. Clone the repo

```sh
git clone https://github.com/Clevermis/Folding_flower.git
```
